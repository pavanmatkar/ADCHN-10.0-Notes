* Domain Controller

A computer running with active directory services is called Domain Controller
or else it is called as local server if not with active directory

FSMO role -> Flexible Single Master Operation role
it contains 5 roles
1. Schema master
2. Domain naming master
3. pdc
4. RID pool
5. Infrastructure Master

when you create user a sid (Security Identity) is allocated and it is permanent you cannot allocate it to other user even if the original user is deleted

ek domain controller ke saath 500 sid by default hota hai

recycle bin of AD is known as "tombstone memory" it has 90 day validity

Global Catalog (GC) is act as a search engine for our infrastructure or domain controller
