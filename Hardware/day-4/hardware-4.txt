5. Storage Device

two types 1. IDE HDD 2. SATA HDD

IDE HDD also known as PATA HDD

IDE is obsolete now so we will study SATA only

SATA HDD has following parts
1. Plater / disk -> stores data
2. Motor -> rotates plater
3. Actuator -> controls arm movement
4. Arm -> controls the movement of head
5. Head -> reads and writes data on disk 

data is stored in the form of electromagnetic signal

higher the rpm of motor higher is the speed of the HDD e.g. 5400rpm, 7200rpm, 10000rpm, etc.

* SSD
no plater
no motor
it has multiple ICs which stores data

if you want data security(recovery) go for HDD
if you want faster speed go for SSD

* MBR (master boot recorder) and GPT (graphic partition table)

MBR supports up to 3 primary partition and one extended partition (logical volume) total of 4 partition and maximum disk size of 2TB

MBR introduced by IBM in 1983

MBR works on old technology (BIOS) and GPT works on new technology (UEFI)

OS can be installed only in primary partition

MBR does not support secure boot menu. GPT supports secure boot menu.

GPT supports up to 128 primary partition
GPT supports UEFI boot menu (unified extendable firmware interface) 

Server uses GPT disks
