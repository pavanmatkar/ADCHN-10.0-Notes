* Microsoft 365 administrator interview questions and answers

MS 365 is a SAAS cloud service

=====================================

Q. diff between MS 365 and office 365

MS 365 include office 365 plus windows OS, enterprises mobility and security

MS 365 is advanced version of office 365

advanced security options are available in MS 365

======================================

Q. what are the admin roles in MS 365

global admin -> all rights

billing admin -> billing related rights

exchange admin manages the mail service e.g. encryption, mail trace karna
ho, restrictions on mail box.

share point admin center and team admin center are collaboration and communication tools.

security admin and compliance admin is for conditional access policy creation, retention policy, etc.
